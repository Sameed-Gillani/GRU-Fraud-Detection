# Real-Time Fraud Detection in Payment Streams Using Gated Recurrent Units

## Overview
A GRU-based deep learning model for real-time fraud detection 
in digital payment streams using the PaySim synthetic dataset.

## Results
- Accuracy: 96%
- Precision (Fraud): 0.95
- Recall (Fraud): 0.96
- F1-Score (Fraud): 0.96

## Files
- `GRU_Fraud_Detection.ipynb` — Full model code

## Dataset
Download PaySim from Kaggle:
https://www.kaggle.com/datasets/ealaxi/paysim1

- Python 3.10
- TensorFlow / Keras
- Scikit-learn
